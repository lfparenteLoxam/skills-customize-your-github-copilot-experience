
# 📘 Assignment: Games in Python - Hangman

## 🎯 Objective

Build a beginner-friendly Hangman game in Python to practice core programming skills such as loops, conditionals, string manipulation, and user input handling.

## 📝 Tasks

### 🛠️ Build the Core Hangman Game

#### Description
Create a playable Hangman game where a player guesses letters to uncover a hidden word. The game should continue until the word is guessed or attempts run out.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list.
- Display the word progress using underscores (for example: `_ _ _ _`).
- Ask the player to guess one letter at a time.
- Reveal correctly guessed letters in all matching positions.
- Decrease remaining attempts after incorrect guesses.
- End the game with a clear win or lose message.

### 🛠️ Improve Game Experience

#### Description
Enhance your game so it is easier to play and gives students clearer feedback while learning.

#### Requirements
Completed program should:

- Prevent repeated guesses from reducing attempts.
- Show the letters that have already been guessed.
- Validate input so only a single alphabetic character is accepted.
- Display the hidden word when the player loses.
